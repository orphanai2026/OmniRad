ChatGPT Bridge

Send prompts to ChatGPT in one click

Drag the button below to your browser's bookmarks bar. Then while you're on ChatGPT, click it to paste the last prompt you generated in Prompt Studio — no copy/paste needed.

01 Install the bookmarklet

Drag this button up to your bookmarks bar (make sure it's visible: press Ctrl+Shift+B if hidden).

📎 Send to ChatGPT
⚠️ Don't click yet — drag to the bookmarks bar first.

02 How it works

STEP A
Studio → Generate prompt
Open Prompt Studio, build your prompt (Preset / Case Builder / Custom), then click Copy for ChatGPT.
STEP B
Open ChatGPT
Go to chatgpt.com and start a new chat (or open any existing one).
STEP C
Click the bookmarklet
Click 📎 Send to ChatGPT in your bookmarks bar. The prompt is pasted automatically.
STEP D
Press Enter
ChatGPT generates the image. Right-click → Save image, then upload via Bulk Upload.
Tip: Studio's Copy for ChatGPT button stores the prompt in this browser. The bookmarklet reads that same store. As long as you use the same browser, the flow works offline.

03 Troubleshooting

  1. «No prompt saved yet» — go to Prompt Studio first and click Copy for ChatGPT before using the bookmarklet.
  2. Nothing happens on ChatGPT — the composer input may not be visible. Scroll to the bottom of the chat so the input box is on screen, then click again.
  3. Different browser — the prompt is stored per-browser. Set it up on each device separately.
  4. ChatGPT UI changed — if OpenAI redesigns the input, the bookmarklet will need an update. Come back to this page for the latest version.
Legal note: The bookmarklet does not scrape data from or automate ChatGPT beyond pasting a single prompt into its native input box. It's a browser-side convenience, equivalent to Ctrl+V. Every image is manually generated and reviewed before entering the atlas.
