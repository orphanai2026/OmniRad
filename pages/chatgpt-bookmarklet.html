02 How it works

STEP A
Studio → Generate prompt
Open Prompt Studio, build your prompt, then click Copy EN or Copy AR.
STEP B
Open ChatGPT
Go to chatgpt.com and start a new chat (or open any existing one).
STEP C
Click the bookmarklet
Click Send to ChatGPT in your bookmarks bar. The prompt is pasted automatically.
STEP D
Press Enter → Upload
ChatGPT generates the image. Save it, then upload via Bulk Upload.
i
Tip: Copy the prompt in Studio first — the bookmarklet reads it from your clipboard. As long as you use the same browser session, the flow works offline.

03 Troubleshooting

  1. No prompt in clipboard — go to Prompt Studio first and click Copy EN / Copy AR before using the bookmarklet.
  2. Nothing happens on ChatGPT — scroll to the bottom so the input box is visible, then click again.
  3. Different browser — install the bookmarklet on each device separately.
  4. ChatGPT UI changed — if OpenAI redesigns the input, come back here for the latest version.
!
Legal: The bookmarklet only pastes a single prompt into the native input — equivalent to Ctrl+V. No scraping, no automation beyond that. Every image is generated and reviewed manually before entering the atlas.
