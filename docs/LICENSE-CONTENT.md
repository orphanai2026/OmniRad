## ٢) شروط الاستخدام (Terms of Use)
- المنصّة **أداة تعليمية بحتة** — الاستخدام السريري أو التشخيصي محظور صراحةً (راجع `docs/IP.md` §5).

- الحذف يخضع لسياسة Soft-Delete (٣٠ يوماً مهلة استرجاع) قبل الحذف النهائي.

## ٣) اتفاقية المساهمين (Contributor License Agreement — CLA)
برفع أي صورة أو مادة عبر Bulk Upload / Contribute Hub، يقرّ المساهم بما يلي:
1. المادة المرفوعة من إنتاجه أو يملك حقّ رفعها.
2. يمنح منصّة OmniRad **ترخيصاً غير حصري، دائماً، عالمياً** لعرض المادة وتوزيعها ضمن الأطلس التعليمي بعد اعتمادها عبر طابور المراجعة.
3. يحتفظ بحقّ النسب (يظهر اسمه كمساهم عند العرض إن اختار ذلك).
4. المنصّة تحتفظ بحقّ الرفض أو الأرشفة (Rejected Archive) دون حذف نهائي.

## ٤) سياسة الخصوصية المختصرة
- لا تُعرَض مفاتيح الخدمة الحسّاسة (`service_role`) في الواجهة الأمامية — anon key فقط.
- الأسرار (Resend، fal.ai، إلخ) محفوظة في Supabase Vault.
- RLS مفعَّل على كل جدول بيانات.
