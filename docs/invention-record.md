# مستند توثيق الابتكار (Invention Disclosure Record)

> صاحب الابتكار: **محمد سعيد الزهراني** · تاريخ التوثيق: 19 يوليو 2026 · منصّة OmniRad

هذا المستند يثبّت تاريخ ووصف الابتكارات المنهجية في منصّة OmniRad، تمهيداً لأي نشر علمي (RSNA/RadioGraphics) أو تقييم لاحق لأهلية براءة الاختراع. لا يُنشَر علنياً قبل استشارة قانونية.

## ١) نظام OSERN — توحيد التسمية التشريحية/الإشعاعية

**المشكلة التي يحلّها:** لا يوجد نظام واحد يوحّد بين المصطلح التشريحي الكلاسيكي (TA2)، والمصطلح الإشعاعي (RadLex/RSNA)، وتصنيف DICOM للمنطقة المصوَّرة (CID 4031)، ومركّب عربي مطابق دلالياً — بل تتعامل الأدوات القائمة (Radiopaedia، IMAIOS e-Anatomy) مع كل معيار بمعزل عن الآخر.

**آلية العمل:**
1. **طبقة التطبيع (`REGION_CANON`):** تُطبِّع أي تسمية منطقة تشريحية واردة (قديمة/مختلفة الصياغة) إلى تسع فئات قياسية معتمدة من DICOM CID 4031 + LOINC/RSNA Playbook "Region Imaged"، وقت العرض (runtime)، دون الحاجة لتنظيف البيانات المخزَّنة مسبقاً.
2. **المُرَكِّب (Composer):** يُخرِج تلقائياً `body_part_examined` + `series_technique` + `region` + `modality` متوافقة مع DICOM من مدخلات وصفية بسيطة (حقلين فقط يُدخلهما المساهم).
3. **الطبقة العربية:** مصطلح عربي مطابق دلالياً (لا ترجمة حرفية) لكل مُدخل، مبنيّ على مزيج Wikidata (سرعة) + مراجعة WHO UMD (دقّة نهائية) — نهج هجين موثّق.
4. **الإثراء التلقائي:** كل بنية تشريحية تُخصَّب تلقائياً بمعرّف RadLex عبر BioPortal API عند الإدخال، مع شارة "Verified" عند التطابق.

**الحداثة المحتملة:** لا يوجد — بحسب البحث المتاح وقت التوثيق — نظام منشور يجمع هذه الطبقات الأربع (TA2 + RadLex + DICOM CID + عربي مُركَّب) في تدفّق تلقائي واحد لمنصّة تعليمية.

## ٢) AI Coherence Score — تقييم اتساق الصور الطبية المولَّدة

**المشكلة:** الصور التعليمية المولَّدة بالذكاء الاصطناعي (لا مصدر DICOM خام) لا تملك معياراً موضوعياً للتحقّق من الدقّة التشريحية عبر المودالتي المتعددة لنفس البنية (مثال: هل مقطع MRI T2 لنفس العضو متّسق تشريحياً مع مقطع CT له؟).

**آلية العمل (3-Layer Quality):**
- **الطبقة الأولى (Auto):** تحقّق آلي من الاتساق البنيوي (الموضع التشريحي، التماثل، غياب أخطاء واضحة).
- **الطبقة الثانية (Admin):** مراجعة بشرية عبر طابور المراجعة (Review Queue) قبل الاعتماد النهائي.
- **الطبقة الثالثة (Community Flagging):** إبلاغ من المستخدمين عن تناقضات لاحقة، يُعاد فحصها.

**الحداثة المحتملة:** منهجية تقييم ثلاثية الطبقة مخصّصة لصور تعليمية مولَّدة (لا حقيقية) عبر مودالتي متعددة — تصنيف مبدئي: **يُنشر علمياً** أولى من طلب براءة، نظراً لطابعها المنهجي/الإجرائي.

## ٣) معمارية الأطلس متعدّد المودالتي ثنائي اللغة (Unified Lens)

عرض عضو تشريحي واحد عبر مودالتي متعددة (X-Ray · CT · MRI · US · PET) جنباً إلى جنب، بلغتين متزامنتين، مع مقارنة (RadCompare) متعدّدة الألواح متزامنة المؤشّرات — مبني فوق طبقة OSERN أعلاه.
